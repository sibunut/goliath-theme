<p align="center">
  <b>Goliath</b> — это экзотическая и гармоничная тема для Visual Studio Code с уникальной пастельной палитрой.
</p>

Мотивация создания данной темы было то, что в моей стране заблокировали возможность скачивания расширений из моего любимого редактора Visual Studio Code, но установив Codium обнаружил недостаток нормальных тем (я использовал One Monokai Pro). Так и не найдя ничего нормального, решил создать свою тему на базе One Monokai, совместив её с вайбами Breeze и Gruvbox, добавив уюта от себя. Тема Goliath совмещает в себе уют, читабельность и комфорт для глаз.

## 📦 Установка

### Из Marketplace (рекомендуется)

1. Откройте VSCode
2. Нажмите `Ctrl+Shift+X`
3. Найдите **Goliath Theme**
4. Нажмите **Install**
5. Нажмите `Ctrl+K Ctrl+T` и выберите `Goliath Dark` или `Goliath Light`

### Ручная установка

```bash
git clone https://github.com/sibunut/goliath-theme.git
cd goliath-theme
npm install -g @vscode/vsce
vsce package
code --install-extension goliath-theme-2.1.0.vsix
```
